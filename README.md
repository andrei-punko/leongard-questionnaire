# Опросник Леонгарда - Акцентуации характера

[![pages-build-deployment](https://github.com/andrei-punko/leongard-questionnaire/actions/workflows/pages/pages-build-deployment/badge.svg)](https://github.com/andrei-punko/leongard-questionnaire/actions/workflows/pages/pages-build-deployment)
[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)

Веб-версия характерологического опросника К. Леонгарда (88 вопросов, 10 шкал акцентуаций). Ответы обрабатываются локально в браузере.

## Запуск онлайн

Откройте [опросник онлайн](https://andrei-punko.github.io/leongard-questionnaire/) или

## Запуск локально

Откройте локальный [`index.html`](index.html) в браузере.

При необходимости можно поднять простой сервер:

```bash
npx serve .
```

## Как пользоваться

1. Прочитайте инструкцию на странице
2. Ответьте на все 88 вопросов: **Да** или **Нет**
3. Нажмите **Показать результат**
4. В блоке результатов - баллы по 10 шкалам и уровень выраженности
5. Ниже - справочная таблица с описанием акцентуаций из методички

Кнопка **Пройти заново** сбрасывает ответы и скрывает результаты.

## Интерпретация баллов

После подсчёта совпадений с ключом сумма по каждой шкале умножается на коэффициент (×2, ×3 или ×6):

| Баллы | Уровень                 |
|-------|-------------------------|
| 0–12  | Норма (слабо выражено)  |
| 13–18 | Тенденция к акцентуации |
| 19–24 | Выраженная акцентуация  |

Максимум по шкале - 24 балла.

## Шкалы

1. Гипертимность (×3)
2. Возбудимость (×2)
3. Эмотивность (×3)
4. Педантичность (×2)
5. Тревожность (×3)
6. Циклотимность (×3)
7. Демонстративность (×2)
8. Неуравновешенность (×3)
9. Дистимность (×3)
10. Экзальтированность (×6)

## Источник

Тексты вопросов, ключ и описания взяты из PDF-методички в корне репозитория.

## Ограничения

Опросник предназначен для самодиагностики и ознакомления. Результаты не сохраняются и не отправляются на сервер. Не заменяет консультацию психолога или психиатра.
